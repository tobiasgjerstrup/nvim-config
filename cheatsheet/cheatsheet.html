Neovim Cheatsheet

Leader key: <Space>

Custom Keybindings

File Management (Telescope)

<leader>ffFind files
<leader>fgLive grep
<leader>fbFind buffers
<leader>fhHelp tags
<leader>fdFind diagnostics
<leader>fqDiagnostics to quickfix list

Buffer Navigation

S-eNext buffer
S-qPrevious buffer
S-wDelete buffer
C-wForce delete buffer

File Tree (nvim-tree)

C-bToggle file tree
CROpen file (focus stays on tree)

Workspaces

<leader>wsSave workspace (prompts for name)
<leader>wlLoad workspace (picker if no name given)
<leader>wdDelete workspace (prompts for name)
<leader>wnList all workspaces
<leader>w1 … <leader>w9Quick-load workspace 1–9
:lua require('config.workspaces').save("name")
:lua require('config.workspaces').load("name")
:lua require('config.workspaces').delete("name")
:lua require('config.workspaces').list()

Copilot (Insert mode)

C-lAccept suggestion
C-jNext suggestion
C-kPrevious suggestion
C-xDismiss suggestion

Completion — CMP (Insert mode)

TabNext completion item
S-TabPrevious completion item
CRConfirm selection

LSP

gdGo to definition
gDGo to declaration
giGo to implementation
grGo to references
KHover (documentation)
[d / ]dPrevious / next diagnostic
[e / ]ePrevious / next error
[w / ]wPrevious / next warning

Editing (Insert mode)

C-hDelete previous word
C-BSDelete previous word
C-BackspaceDelete previous word

Built-in Neovim Essentials

Cursor Movement

h / j / k / lLeft / down / up / right
w / bNext / previous word start
e / geNext / previous word end
0 / ^Start of line (first column / first non-blank)
$End of line
gg / GStart / end of file
5j / 5kJump 5 lines down / up
{ / }Previous / next paragraph
%Jump to matching bracket

Editing

i / IInsert at cursor / start of line
a / AAppend after cursor / end of line
o / ONew line below / above
xDelete character under cursor
dd / 3ddDelete line / 3 lines
dw / dawDelete word / delete a word
DDelete to end of line
JJoin line below to current
~Toggle case
r + charReplace character under cursor
.Repeat last change

Copy & Paste

yy / YYank (copy) line
3yyYank 3 lines
ywYank word
y$Yank to end of line
p / PPaste after / before cursor
"ayyYank into register a
:regView all registers

Visual Mode

vVisual mode (character)
VVisual mode (line)
C-vVisual block mode
> / <Indent / dedent selection
=Auto-indent selection

Undo & Redo

uUndo
C-rRedo

Searching

/patternSearch forward
?patternSearch backward
n / NNext / previous match
* / #Search word under cursor
fcharFind char forward on line
FcharFind char backward on line
tcharFind before char forward
TcharFind before char backward
; / ,Repeat last f/t forward / backward

Find & Replace

:%s/old/new/gReplace all in file
:%s/old/new/gcReplace all in file (confirm each)
:s/old/new/gReplace all on current line
:'<,'>s/old/new/gReplace in visual selection

Splits & Windows

:sp / :vspHorizontal / vertical split
C-w h/j/k/lMove to left/down/up/right window
C-w v / C-w sVertical / horizontal split
C-w qClose window
C-w oClose all other windows

Macros

q + registerStart recording (e.g. qa)
qStop recording
@ + registerReplay macro (e.g. @a)
@@Replay last macro

Marks

m + {a-z}Set local mark
m + {A-Z}Set global mark
' + markJump to mark (line)
` + markJump to mark (exact position)

Misc

:wSave file
:qQuit
:wq / :xSave and quit
:q!Quit without saving
:e fileOpen file
:bdDelete buffer
Ctrl-6 / :b#Toggle to previous buffer
